# CN-Project

Secure Multi-Client Chat System

Description:
This project implements a secure chat system using TCP sockets. The server supports multiple clients using threading and ensures secure communication using SSL.

Features:
- Multi-client support
- Real-time messaging
- SSL/TLS encryption
- Username-based chat

How to run:
1. Run server.py
2. Run multiple instances of client.py
3. Enter usernames and start chatting

Performance:
Tested with multiple clients and handled concurrent communication efficiently.

Future Improvements:
- Multi-room chat support
- File transfer

  
Requirements:
- Python 3.x
